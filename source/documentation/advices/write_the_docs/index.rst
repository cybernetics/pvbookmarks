Write the docs vision
======================

.. seealso:: http://docs.writethedocs.org/en/2013/about/vision.html 

**Documentation is how you share your creations with the world.**

If you want people to benefit from your work, they have to be able to 
use it. 

Help me, help you. Do something and change something.

We believe it should be easy for people to start writing documentation. 
There should be straight-forward guides to getting started with good tools.
