Getting Started
================

JSDoc 3 is an API documentation generator for JavaScript, similar to JavaDoc or 
PHPDoc. 

You add documentation comments directly to your source code, right along side 
the code itself. 

The JSDoc Tool will scan your source code, and generate a complete HTML 
documentation website for you.
